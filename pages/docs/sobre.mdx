---
title: Sobre o Projeto
---

Este é um template de projeto de SaaS de autenticação e autorização implementado em [Next.js](https://nextjs.org/) e [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).
Foi incluído a inicialização com [Shadcn-ui](https://ui.shadcn.com/), prisma [Prisma](https://www.prisma.io/), Authjs | Next-Auth [Authjs](https://authjs.dev/) utilizando banco de dados [PostgreSQL](https://www.postgresql.org/)

Além das functionalides básicas de segurança, está incluso neste Starter Kit functionalidade como:

| Nome | Tipo | Descrição |
| ---- | ---- | ----------- |
| `Multi-Step-Form` | Funcionalidade | Funcionalidade de processo guiado |
| `Website Builder` | Funcionalidade | Construção de paginas no-code |
